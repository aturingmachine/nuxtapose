    # CLI Options

    These are options that can be provided to any nuxtapose command.

    # Logging Levels

    • --debug, -d
      • increases verbosity of logging
    • --quiet, -q
      • supress all logs and after run reports
    • --verbose, -v
      • equal to --debug and "reportLevel": "all"
